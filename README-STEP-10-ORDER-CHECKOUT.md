# FRZN Alpine Fashion UI — Step 10 Order Checkout

This patch inserts a premium add-to-cart and customer order section immediately
after the updated Step 08 editorial section and before the Step 09 footer.

## Included

- Updated `index.html`
- Updated modular `css/index.css`
- `css/sections/order-form.css`
- `js/order-form.js`

## Front-end demo behaviour

- Cart quantity increase, decrease and remove
- Dhaka / outside-Dhaka delivery charge switching
- District-based automatic delivery selection
- Promo code demo: `FRZN10`
- Grand-total calculation
- Customer-form validation
- Invoice preview modal
- Navbar bag and hero CTA scroll to checkout

## Install

Extract this ZIP directly into the project root and allow overwrite.
